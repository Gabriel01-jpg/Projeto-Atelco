Exercício 04

Questão 04. Faça um algoritmo que leia a idade de uma pessoa expressa em dias e mostre-a expressa em anos, meses e dias..

O código ficou dessa forma: const idadeEmDias = 7328;
const anos = Math.floor(idadeEmDias / 365);
const meses = Math.floor((idadeEmDias % 365) / 30);
const dias = ((idadeEmDias % 365) % 30);
console.log(`${anos} anos, ${meses} meses e ${dias} dias`);
Resultado: 20 anos, 0 meses e 28 dias
