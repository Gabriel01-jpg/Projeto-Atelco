Questão 10 - Elaborar um algoritmo que lê 3 valores a,b,c e os escreve. A seguir, encontre o maior dos 3 valores e o escreva com a mensagem: "É o maior ".

O código ficou dessa forma: let valores = { A: 35, B: 15, C: 20 }
let maior = -999999;
for (valor in valores){
console.log(valores[valor])
if(valores[valor] > maior){
maior = valores[valor]
}
}
console.log('\n O maior valor é ' + maior)
Resultado: O maior valor é 35;
