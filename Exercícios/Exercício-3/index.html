Exercício 03

Questão 03. Faça um algoritmo que leia a idade de uma pessoa expressa em anos, meses e dias e mostre-a expressa apenas em dias.

O código ficou dessa forma: const valoresIdade = { ano: 25, meses: 5, dias: 10 }; const anoBissexto = Math.floor(valoresIdade.ano / 4) const valoresemDias = ((valoresIdade.ano * 365) + anoBissexto) + (valoresIdade.meses * 30) + valoresIdade.dias console.log(valoresemDias + 'dias')
Resultado: 9291 dias
